How TechCure Works

  • Sensor-Driven Monitoring - ESP32 with flow and pulse sensors monitors IV flow rate and vitals.
  • Real-Time Alerts - System triggers buzzer and LED indicators if abnormal flow or saline depletion is detected.
  • Emergency Notifications - Automatic alerts are sent to nurses and backend when emergencies occur.
  • Backend Integration - Sends data to a secure server for logging, analysis, and dashboard visualization.
  • Simulation Support - Simulated mode for safe testing without physical hardware.
